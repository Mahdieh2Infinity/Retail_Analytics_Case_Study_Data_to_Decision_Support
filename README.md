# Retail Analytics Case Study – From Data to Decision Support

This portfolio project demonstrates an end-to-end retail analytics workflow using synthetic retail data.

The project includes:
- Synthetic dataset creation
- Demand forecasting
- Promotion effectiveness analysis
- Store performance evaluation
- Scenario simulator
- Executive decision-support recommendations

## Tools Used
- Python
- Pandas
- Machine Learning
- Forecasting
- Data Visualization

## Project Files
- Full project report: `Mahdieh_Retail_Analytics_Case_Study_Data_to_Decision_Support.pdf`
- Python notebook: `Retail_Analytics_Case_Study.ipynb`
- Synthetic datasets: `Synthetic_Retail_Datasets.zip`

## Key Business Questions
- How can we forecast seasonal demand more accurately?
- Which promotions improve profitability?
- Why do some stores outperform others?
- How can scenario simulation support executive decision-making?

## Project Highlights
- Built realistic synthetic retail datasets
- Designed forecasting and promotion analytics workflows
- Evaluated store performance using business KPIs
- Developed decision-support recommendations for executives
